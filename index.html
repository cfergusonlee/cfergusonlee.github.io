View Github Repo

This project identified which Enron employees were more likely to have committed fraud from public Enron financial and email data using machine learning. Uber Exploratory Analysis

View Github Repo

This project uses R to explore the variables, structure, patterns, oddities, and underlying relationships of New York City Uber pickups from January through June of 2015. The dataset consisted of 14 million observations spread across 12 variables. Exploratory analysis revealed the cyclical nature of rider behavior and that population density, not population size contributed most to trip frequency.

UFC Fight Prediction

View Github Repo

This project predicts the winner of a UFC fight with 88% accuracy. All data was scraped from fightmetric.com using BeautifulSoup, generating a total of 32 features and 8772 observations. Multiple algorithms are tested, including Perceptron, Random Forests, Decision Trees, SGD, Linear SVC, Gaussian Naive Bayes and K-Nearest Neighbors. It was a lot of fun putting this project together and development will continue over time.

Iceberg Image Classifier

View Github Repo

This project processes synthetic aperture radar images in Python and trains a machine learning model to distinguish icebergs from ships. The dataset was a JSON file consisting of 1604 observations. Each observation contained two flattened polarized images, an id, the incident angle used to capture the image, and a label. Averaging input vectors from each of the horizontal and vertical polarized image bands led to 75% accuracy.

Authorship Identification

View Github Repo

This notebook tutorial demonstrates how to use natural language processing to identify authors from text excerpts. The dataset is comprised of 19,579 records and 3 variables: id, text and author. Stemming, word counts and lemmatization are used to engineer new features and a variety of classifiers are tested. Multinomial naive bayes yielded the highest results, accurately identifying the author 85% of the time.
