UFC Fight Predictor

View Github Repo

This project predicts the winner of a UFC fight with 87% accuracy. All data was scraped from fightmetric.com using BeautifulSoup, generating a total of 32 features and 8772 observations. Multiple algorithms are tested, including Perceptron, Random Forests, Decision Trees, SGD, Linear SVC, Gaussian Naive Bayes and K-Nearest Neighbors. It was a lot of fun putting this project together and development will continue over time.
